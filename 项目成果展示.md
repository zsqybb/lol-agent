
# 二、项目成果展示

## 2.1 产品概述

### 产品名称
**《英雄联盟》智能问答系统**

### 产品创意
本项目创新性地整合三大核心技术能力，为《英雄联盟》玩家提供一站式游戏信息服务：
- **Riot官方API**：获取真实玩家数据（段位、战绩、熟练度）
- **LCU本地客户端**：读取本机账号信息，无需登录
- **讯飞星火大模型**：智能问答，提供英雄攻略、背景故事等

### 应用场景
| 场景 | 描述 | 目标用户 |
|------|------|----------|
| 玩家查询 | 查询任意玩家的段位、战绩、英雄熟练度 | 所有玩家 |
| 本机信息读取 | 一键读取当前登录账号信息 | 个人用户 |
| 智能问答 | 获取英雄攻略、出装建议、背景故事 | 新手玩家 |
| 英雄图鉴 | 浏览所有英雄数据和技能信息 | 全玩家 |

### 功能三：智能问答系统

**功能描述**：支持三种问答模式

**模式一：AI问答**
- 自由问答，支持闲聊和游戏相关问题
- 示例："英雄联盟有多少个英雄？"

**模式二：英雄问答**
- 专注于英雄攻略、出装、符文
- 示例："亚索怎么玩？"

**模式三：故事问答**
- 英雄背景故事、阵营关系
- 示例："德玛西亚有哪些英雄？"

## 2.4 项目部署

### 本地运行方式

1. **环境要求**：
   - Python 3.8+
   - 安装依赖：`pip install -r requirements.txt`

2. **配置API密钥**：
   - 在设置页面配置Riot API密钥和讯飞星火API密钥

3. **启动服务**：
   ```bash
   python web_server.py
   ```

4. **访问地址**：http://127.0.0.1:5000
